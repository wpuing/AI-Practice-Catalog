# 提交操作手册和截图文件的步骤

## 问题
操作手册（OPERATION_MANUAL.md）和截图文件（screenshots/*.png）没有被添加到Git暂存区。

## 解决方案

### 方法1：使用脚本（推荐）

**Windows (PowerShell):**
```powershell
cd AI-Practice-Catalog
.\添加文件并提交.ps1
```

**Windows (CMD):**
```cmd
cd AI-Practice-Catalog
添加文件并提交.bat
```

### 方法2：手动执行命令

```bash
cd AI-Practice-Catalog

# 1. 添加操作手册
git add template-project/OPERATION_MANUAL.md

# 2. 添加所有截图文件
git add template-project/screenshots/*.png

# 3. 添加截图README
git add template-project/screenshots/README.md

# 4. 添加其他修改的文件
git add -A

# 5. 检查状态（应该看到所有文件都在暂存区）
git status

# 6. 提交
git commit -F COMMIT_MSG.txt

# 7. 推送到远程仓库
git push origin master
```
