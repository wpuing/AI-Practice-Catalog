# API文档索引

本目录包含项目的API接口文档。

## 📡 API文档

### API.md - 完整API接口文档

包含所有REST API接口的详细说明：

- **统一响应格式**: Result<T> 结构说明
- **接口列表**: 所有可用的API接口
- **请求示例**: 各种请求的JSON示例
- **响应示例**: 各种响应的JSON示例
- **错误码说明**: 所有状态码的含义
- **使用示例**: 实际调用示例

## 🔑 主要接口模块

- **认证授权**: 登录、注册、Token刷新、退出登录
- **用户管理**: 用户CRUD操作
- **角色管理**: 角色CRUD和权限分配
- **商品管理**: 商品和商品类型的完整管理
- **安全配置**: 白名单和权限配置管理
- **系统管理**: Redis管理、测试工具

## 📝 使用说明

1. 查看 `API.md` 了解所有接口详情
2. 参考请求/响应示例进行接口调用
3. 使用统一的 `Result<T>` 格式处理响应

## 🚀 快速开始

- **接口基础路径**: `/api`
- **认证方式**: Bearer Token (JWT)
- **Content-Type**: `application/json`

## 💡 提示

- 所有接口均返回统一的 `Result<T>` 格式
- 错误信息在 `Result` 的 `message` 字段中
- Token需要在请求头的 `Authorization` 字段中携带
