## 📋 文档分类说明

### 架构文档 (architecture/)
包含项目的架构设计文档，说明分层架构、模块划分、依赖关系等。

### API文档 (api/)
包含所有REST API接口的详细文档，包括接口列表、请求/响应示例、错误码等。

### 重构文档 (refactoring/)
记录项目从平铺式结构重构为分层模块化架构的完整过程，包括：
- 重构指南和步骤
- 迁移进度跟踪
- 清理和验证报告

### 使用指南 (guides/)
包含项目的各种配置和使用指南，帮助开发者快速上手和解决问题。
