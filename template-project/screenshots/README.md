## 截图说明

所有截图文件都是PNG格式，用于 `OPERATION_MANUAL.md` 操作手册文档。

### 截图命名规范

- 使用小写字母和下划线
- 格式：`模块_功能_描述.png`
- 示例：
  - `login_page.png` - 登录页面
  - `users_list.png` - 用户列表
  - `products_add_form.png` - 商品新增表单

## 截图要求

1. **分辨率**：建议使用1920x1080或更高分辨率
2. **格式**：PNG格式（支持透明背景）
3. **大小**：单个文件不超过2MB
4. **内容**：清晰可见，关键信息突出
5. **命名**：使用小写字母和下划线，格式：`模块_功能_描述.png`

## 注意事项

- 截图应包含完整的页面内容，避免截取不完整
- 敏感信息（如真实用户名、密码）应打码处理
- 保持截图风格一致，使用相同的浏览器和分辨率
- 定期更新截图，确保与系统界面保持一致
